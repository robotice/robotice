Robotice monitoring
===============

Opensource monitoring, reasoning and acting framework.

`Website`_

`Video demostration`_

Usage
-----

App for distributed monitoring small devices like a BeagleBone black or Raspberry Pi.

Supported sensors:
-----

* `DHT11`_ / `DHT2302`_
* `TMP36`_
* `Hygro soil`_
* `relay_board`_
* `Sispm`_


Requirements
-----

* celery
* database - now only redis is supported
