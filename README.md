
# Robotice monitoring daemon

Application is in /srv/robotice/service.

Configuration is in /srv/robotice/config.

## Usage

All services are run using supervisor daemon.

## Requirements

* PyYAML 

Logs in /srv/robotice/logs/service_xxx.log
